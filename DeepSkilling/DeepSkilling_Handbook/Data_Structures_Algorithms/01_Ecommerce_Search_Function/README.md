# E-commerce Search Function

## Objective

Implement a product search feature using Linear Search.

## Description

This program searches for a product in an e-commerce catalog based on the product name.

## Files

* EcommerceSearch.java

## Expected Output

Product Found:
ID: 102
Name: Phone
Category: Electronics

## Concepts Used

* Arrays
* Classes and Objects
* Linear Search
* Java Methods

## Benefits

* Simple searching technique
* Easy to implement
* Useful for small datasets
